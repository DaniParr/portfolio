Underwater Image Enhancement with AI

In order to improve the effectiveness of the Machine Intelligence Lab's autonomous underwater vehicle (AUV), Subjugator, it needs to be able to make accurate distinctions in its environment. With water refracting light, it can be difficult to use the raw image from the camera to distinct the types of objects in view for the UAV. Utilizing the machine learning model provided by Xuelei Chen, Pin Zhang, Lingwei Quan, Chao Yi, Cunyue Lu from their paper Underwater "Image Enhancement based on Deep Learning and Image Formation Model," we were able to successfully drive data from one of the camera topics to be enhanced and be constantly output for clearer imaging.

Features

Image Enhancement through ROS

In order to connect the camera topic to the image enhacnement node, we utilized the Robot Operating System frame work to subscribe to the camera topic. From there we fed a constant stream of images from the camera to be processed.
